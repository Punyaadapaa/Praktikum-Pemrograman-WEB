# Praktikum Pemrograman Website 🌐

Repositori ini berisi kumpulan tugas praktikum mata kuliah Pemrograman Website Semester 4 - Informatika UNS.

## 📂 Daftar Isi

### 1. [Praktikum Ke-2: Halaman Login](./Praktikum%202)

- **Tugas**: Membuat desain halaman **LOGIN** yang kreatif.
- **Teknologi**: HTML5 & CSS3.
- **Fitur**: Form input user, password, dan styling kustom.

### 2. [Praktikum Ke-5: JavaScript & AJAX](./Praktikum%205)

- **Tugas**: Pengembangan dari tugas pertama dengan penambahan fitur interaktif.
- **Teknologi**: HTML, CSS, JavaScript (DOM & AJAX).
- **Fitur**:
  - Penambahan form **Register**.
  - Implementasi **JavaScript & DOM** untuk interaktivitas.
  - Penggunaan **AJAX** dengan method `GET` dan `POST` (menggunakan jQuery atau JS murni).

### 3. [Praktikum Ke-6: Aplikasi CRUD & Database MySQL](./Praktikum%206)

- **Tugas**: Membuat aplikasi manajemen inventaris "Toko Komputer" (Smartbuild) berbasis web dengan integrasi database[cite: 1].
- **Teknologi**: PHP, MySQL, HTML, CSS, JavaScript.
- **Fitur**:
  - Implementasi operasi **CRUD (Create, Read, Update, Delete)** untuk mendata komponen komputer.
  - Fitur **Upload Gambar** produk yang akan disimpan secara otomatis ke dalam direktori `/uploads/`.
  - Integrasi database MySQL menggunakan file `db.php` beserta struktur tabel bawaan dari `smartbuild_db.sql`.
