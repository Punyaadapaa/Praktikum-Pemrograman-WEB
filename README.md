# Praktikum Pemrograman Website 🌐

Repositori ini berisi kumpulan tugas praktikum mata kuliah Pemrograman Website Semester 4 - Informatika UNS.

## 📂 Daftar Isi

### 1. [Praktikum Ke-2: Halaman Login](./Praktikum%202)

- **Tugas**: Membuat desain halaman **LOGIN** yang kreatif.
- **Teknologi**: HTML5 & CSS3.
- **Fitur**: Form input user, password, dan styling kustom.

### 2. [Praktikum Ke-5: JavaScript & AJAX](./Praktikum%205)

- **Tugas**: Pengembangan dari tugas pertama dengan penambahan fitur interaktif.
- **Teknologi**: HTML, CSS, JavaScript (DOM & AJAX).
- **Fitur**:
  - Penambahan form **Register**.
  - Implementasi **JavaScript & DOM** untuk interaktivitas.
  - Penggunaan **AJAX** dengan method `GET` dan `POST` (menggunakan jQuery atau JS murni).

### 3. [Praktikum Ke-6: Aplikasi CRUD & Database MySQL](./Praktikum%206)

- **Tugas**: Membuat aplikasi manajemen inventaris "Toko Komputer" (Smartbuild) berbasis web dengan integrasi database.
- **Teknologi**: PHP, MySQL, HTML, CSS, JavaScript.
- **Fitur**:
  - Implementasi operasi **CRUD (Create, Read, Update, Delete)** untuk mendata komponen komputer.
  - Fitur **Upload Gambar** produk yang akan disimpan secara otomatis ke dalam direktori `/uploads/`.
  - Integrasi database MySQL menggunakan file `db.php` beserta struktur tabel bawaan dari `smartbuild_db.sql`.

### 4. [Praktikum Ke-9: PHP Native OOP, CRUD, Session & Cookies](./Praktikum%208)

- **Tugas**: Membuat website dengan _pure_ PHP Native berbasis _Object-Oriented Programming_ (OOP) dilengkapi validasi halaman.
- **Tema**: **Toko Baju** (Manajemen Inventaris Produk Pakaian).
- **Teknologi**: PHP Native, MySQL, CSS.
- **Fitur**:
  - Pendekatan **OOP Based** penuh dengan pemisahan class `Database` (koneksi) dan class `Produk` (operasi CRUD).
  - Halaman **Login** sebagai validasi akses dengan proteksi _password hashing_ (MD5) sebelum masuk ke halaman utama.
  - Implementasi **Session** (`$_SESSION`) untuk memproteksi seluruh halaman dashboard dan memberikan notifikasi flash message antar halaman.
  - Implementasi **Cookies** (`setcookie`) untuk menyimpan _username_ yang ditampilkan pada topbar dashboard, serta dihapus otomatis saat logout.
  - Desain UI/UX menggunakan kustomisasi **CSS** dengan pendekatan _glassmorphism_ (backdrop-filter, glass card, sidebar transparan).
  - Telah menyertakan file `database.sql` untuk mempermudah _import_ struktur tabel dan data dasar (admin).
